# Week 2 – CSS Animations & Interactions

This week focuses on building interactive and visually engaging user interfaces using CSS animations, transitions, transforms, and basic JavaScript. The goal is to understand how modern websites create smooth user experiences through motion and feedback.

## Topics Covered
- CSS Transitions
- CSS Transforms
- Hover Effects
- Box Shadows
- Pseudo-elements (`::before`, `::after`)
- Background & Border Animations
- Button Animations
- Basic JavaScript Interactions

## Projects

### Project 1 – Animated Buttons

A collection of animated buttons designed to explore different hover and click effects commonly used in modern web design. This project focuses on creating smooth interactions that improve user experience and provide visual feedback.

**Concepts Practiced:**
- Hover state styling
- Smooth transitions
- Transform animations (scale, translate, rotate)
- Shadow and depth effects
- Animated backgrounds and borders
- Pseudo-elements for advanced effects
- Basic click interactions with JavaScript

**Goal:**
To gain hands-on experience with UI animations and understand how small visual effects can make interfaces feel more responsive and polished.

### Project 2 – Animated Cards

A responsive card component project designed to explore different hover animations and visual effects commonly used in modern web interfaces. This project focuses on creating engaging interactions through motion, depth, and smooth transitions while maintaining a clean and responsive layout.

### Concepts Practiced:

- Responsive layouts with Flexbox
- Media queries
- Hover state styling
- Smooth transitions
- Transform animations (translateY, scale)
- Box shadow effects
- Interactive button animations
- Mobile-first design principles

### Goal:

To gain hands-on experience building interactive card components, understand how hover animations improve user engagement, and reinforce responsive design techniques used in modern websites.

### Project 3 – Modern Login Page

A responsive login and signup interface inspired by modern authentication pages. This project focuses on creating smooth transitions between login and signup states while maintaining a clean and user-friendly design across different screen sizes.

### Concepts Practiced

- Responsive layouts with Flexbox
- Media queries
- CSS transitions and transforms
- Form state switching with JavaScript
- Class toggling and DOM manipulation
- Hover and focus effects
- CSS variables and fluid spacing with `clamp()`
- Mobile-first design principles
- Interactive UI animations

### Goal

To gain hands-on experience building a complete responsive form interface, understand how transitions improve user experience, and reinforce the integration of JavaScript with CSS for creating dynamic and interactive web components.

## Learning Outcome

By the end of this week, I will be able to create interactive UI components with smooth animations, understand how motion enhances user experience, and apply modern CSS techniques to build more engaging web interfaces.
