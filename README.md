# Week 2 – CSS Animations & Interactions

This week focuses on building interactive and visually engaging user interfaces using CSS animations, transitions, transforms, and basic JavaScript. The goal is to understand how modern websites create smooth user experiences through motion and feedback.

## Topics Covered
- CSS Transitions
- CSS Transforms
- Hover Effects
- Box Shadows
- Pseudo-elements (`::before`, `::after`)
- Background & Border Animations
- Button Animations
- Basic JavaScript Interactions

## Projects

### Project 1 – Animated Buttons

A collection of animated buttons designed to explore different hover and click effects commonly used in modern web design. This project focuses on creating smooth interactions that improve user experience and provide visual feedback.

**Concepts Practiced:**
- Hover state styling
- Smooth transitions
- Transform animations (scale, translate, rotate)
- Shadow and depth effects
- Animated backgrounds and borders
- Pseudo-elements for advanced effects
- Basic click interactions with JavaScript

**Goal:**
To gain hands-on experience with UI animations and understand how small visual effects can make interfaces feel more responsive and polished.
